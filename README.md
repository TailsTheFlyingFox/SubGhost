## Description
SubGhost is a powerful subdomain discovery tool that helps you uncover hidden or less visible subdomains for a given domain using various public API services. This tool is designed to be user-friendly yet highly flexible, allowing users to choose the output format for the results they receive. Whether you are conducting cybersecurity research, penetration testing, or simply looking to enhance your organization's security posture, SubGhost is a valuable addition to your toolkit.
